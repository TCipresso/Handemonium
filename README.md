"Handemonium" is a video game made by Fistbump Studios for the Pirate Software Game Jam 16.

## Fistbump Studios Developers
- Tommaso Cipresso — Software Engineer  
- Anthony Lira — Software Engineer / Musician  
- Colby Loner — Artist

**Playable Game Link:**  
https://fistbump-studios.itch.io/handemonium

## About
First Person arena shooter with a graphical twist combining 2D and 3D graphics. Meant to test and prototype first person shooter systems, 
and see how 2D art could fair in a 3D environment.
